# Hangzhou Weekend Page

Static GitHub Pages site for a two-day Hangzhou weekend itinerary for two girls.

## Contents

- `index.html` - page structure and itinerary content
- `styles.css` - responsive visual design
- `script.js` - tab switching, reveal animation, and copy action
- `assets/hangzhou-anime-map.png` - generated cute anime-style route illustration

## Local Preview

Open `index.html` directly in a browser, or run a simple static server from this folder.
